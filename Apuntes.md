    #ACID (Atomicity,Consistency,Isolation,Durability)
        Atomicidad: Una transaccion se ejecuta completamente o no se ejecuta.
        Consistencia: La vase de datos pasa siempre de un estado válido a otro válido.
        Aislamiento: Las transacciones concurrentes no interfieren entre sí.
        Durabilidad: Una vez confirmada una transacción, los datos no se pierden.
